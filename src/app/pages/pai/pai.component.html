Passing Information Between Parent and Child Components

In this project, we have a parent component, which is the form, and a child component, which serves as the agenda listing. When adding a new contact, how can information be passed between these two components? A simpler approach would be to use @Input; however, for didactic and learning purposes, this project has employed @ViewChild. With @ViewChild, it becomes possible to access a child component, call methods, or access instance variables available to it, directly from the parent component. It's important to note that in cases where you wish to access multiple children, you should then use @ViewChildren.

Additionally, the following features were utilized in this project:

  • ReactiveForms;
  • Creation of Services;
  • ngAfterViewInit;
